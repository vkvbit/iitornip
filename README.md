# IITORNIP
Is it TOR Node IP (iitornip)


## About
This is a python library for find details about a Tor node via IP address.


## Usage

- Clone the repo
  ```
  git clone https://github.com/vkvbit/iitornip.git
  cd iitornip
  ```
- Install IITORNIP
  ```
  pip install .
  ```
- Update IITORNIP database
  ```
  iitornip -u
  ```
- Run IITORNIP
  ```
  iitornip -i <ip address>
  ```
